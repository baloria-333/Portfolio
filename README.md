# ResuFolio - Resume to Portfolio

Transform your PDF resume into a professional portfolio website in seconds using AI.

## Features
- **PDF Upload**: Drag and drop resume upload.
- **AI Analysis**: Extracts persona, skills, and projects automatically.
- **Interactive Editor**: Real-time preview and editing of your portfolio.
- **Modern UI**: Built with React, Tailwind CSS, and Lucide Icons.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase (Auth & Storage)

## Setup

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Environment Variables**
   Create a `.env` file:
   ```env
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

## Deploy
Push to GitHub and connect to Vercel or Netlify.
